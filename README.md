# Docker Tasks
To run the Docker container with WP + MariaDB:
`docker-compose up`

---

To stop the container:

`docker-compose stop`

---

# Gulp Tasks

To run Gulp with BrowserSync + same Docker localhost:

`gulp`

---

To compile files (include images) and deploy theme:

`gulp deploy`
